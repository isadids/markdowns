# MDN - JavaScript 学习笔记

## 基础语法

### 什么是 JavaScript？

**JavaScript** 是一种**脚本编程语言**。

> **脚本编程语言**，也叫做**解释型语言**，与**编译型语言**相对应：
>
> - **解释型语言**：由**解释器**依次读取、解释并运行每行代码。
> - **编译型语言**：由**编译器**将完整的代码转换成机器指令的形式，最终生成**可执行文件**。

**Javascript** 是一种轻量级的**解释型语言**，而且它是专门用于**客户端浏览器**的，所以也叫**客户端脚本语言**，用于控制网页的 HTML 和 CSS，以实现网页的诸多动态功能。

当然，**JavaScript** 也能在服务端运行，得益于 Node.js。

### 如何向页面添加 JavaScript？

与 CSS 类似，有三种添加 JavaScript 的方式：

- **内联 JavaScript**：通过 HTML 元素属性添加（比如：`<button onclick="createParagraph()">点我！</button>`）。
- **内部 JavaScript**：通过 `<script> ... </script>` 元素包裹 JS 代码来将其嵌入到 HTML 文件中。
- **外部 JavaScript**：通过 `<script src="..."></script>` 引入外部 JS 文件。

> [!WARNING]
>
> **内联 JS**：不推荐使用！因为这不仅**污染**了 **HTML**，而且**效率很低**。

> [!CAUTION]
>
> **内部 JS**：如果要使用**内部 JS**，要确保 `<script>...</script>` 元素位于 `<body>` 的底部，即 `</body>` 之前，这是**主流**且**标准**的做法，因为这样可以确保 **JS** 能够在所有的 **HTML** 加载完成后再执行，**JS** 操作的，总是已经完成加载后的 HTML。 

> [!CAUTION]
>
> **外部 JS**：可以将 **JS** 代码，放在以 `.js` 为后缀的文件中，然后在 **HTML** 中**引用**它。
>
> 有两种推荐的引用方式：
>
> - **在 `<head>` 中**：`<script type="module" src="script.js"></script>` 。
>   然而直接这样做**并不会生效**，这是因为浏览器的**安全机制**。解决方法就是**使用本地 Web 服务器**，也可以使用下面的第二种方法。
> - 同样**在 `<head>` 中**：`<script defer src="script.js"></script>`。
>
> 这涉及到浏览器的**脚本加载策略**。相关内容：[MDN - 脚本加载策略](https://developer.mozilla.org/zh-CN/docs/Learn_web_development/Core/Scripting/What_is_JavaScript#%E8%84%9A%E6%9C%AC%E5%8A%A0%E8%BD%BD%E7%AD%96%E7%95%A5)。

### 变量与常量

JS 中，

- 变量使用关键字 `let` 来声明，例如：`let num;`；
- 常量使用关键字 `const` 来声明，例如: `const sum;`。

### 函数

JS 中，函数使用关键字 `function` 来声明，例如：

```javascript
function checkGuess() {
    alert("I am a placeholdr");
}
```

## 字符串

在 JS 中，**字符串**（**字面量**）可以使用**单引号（`'`）**、**双引号（`"`）**和**反引号（`` ` ``）**来包裹。

> [!TIP]
>
> - **单引号**（`'`）和**双引号**（`"`）字符串是相同的；
> - 但**反引号**（`` ` ``）字符串是一种**特殊的**字符串，叫作**模板字面量**，它具有以下**特性**：
>   - 可以在其中嵌入 JS；
>   - 可以声明多行（在其中直接输入的换行符将被保留）。

### 模板字面量 - 嵌入 JS

在**模板字面量**（反引号 `` ` `` 字符串字面量）中，可以使用 `${ }` 包裹 JS **变量**或**表达式**，以将其嵌入字符串中。
