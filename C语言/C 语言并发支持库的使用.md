## threads.h 简介

*threads.h* 是 C11 及之后的标准提供的用于支持**多线程**开发的标准库的头文件。

## 支持 threads.h 的环境

目前，好像只有 **Linux** 系统下的 **GCC** 编译器**直接**提供了完整的 ***threads.h*** 的支持。

因此，最好在 Linux 平台上使用 *threads.h。*

# 什么是多线程

## 进程的概念

进程，顾名思义，就是正在进行中的程序，所以，正在运行的程序就叫进程。
这是一个非常宽泛的，抽象的定义，或者说广义的定义。

具体来说，程序最开始以可执行文件的形式存储于计算机外部存储器中，当需要运行这个程序时，需要先将程序**加载**到计算机内存中以待执行，这是因为 CPU 与内存交换数据更快。

而当一个程序被加载到内存中并被 CPU 执行时，就称其为一个进程。
