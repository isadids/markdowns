  ```powershell
  Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
  ```



1. 安装 Chocolatey 和 Scoop

   - 安装 Chocolatey

     ```powershell
     Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
     ```

   - 安装 Scoop

     ```powershell
     Set-ExecutionPolicy Bypass -Scope Process -Force; Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
     ```

2. Chocolatey 安装 Cloudfalre Warp 并开启加速

   ```powershell
   choco installl warp  -y --ia 'ORGANIZATION=saibohuofoyyds'
   ```

3. Scoop 安装常用辅助安装工具（Git、7Zip、dark 和 innounp 等）

   ```powershell
   scoop install git 7zip dark innounp aria2 sudo
   ```

4. Scoop 添加常用 `bucket`

   ```powershell
   scoop bucket add extras games nonportable
   ```
