1. 开启 UTF-8；

   | `task`         | `one_command` |
   | -------------- | ------------- |
   | `run_as_admin` | `true`        |

   ```powershell
   Set-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\Nls\CodePage" -Name "ACP" -Value "65001"; Set-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\Nls\CodePage" -Name "MACCP" -Value "65001"; Set-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\Nls\CodePage" -Name "OEMCP" -Value "65001"
   ```

2. 修改脚本执行策略为 `RemoteSigned`（当前用户）；

   | `task`         | `one_command` |
   | -------------- | ------------- |
   | `run_as_admin` | `false`       |

   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```

3. WinGet 安装 Cloudfalre Warp 并开启加速；

   | `task`         | `one_command` |
   | -------------- | ------------- |
   | `run_as_admin` | `true`        |

   ```powershell
   winget install Cloudflare.Warp --custom 'ORGANIZATION=saibohuofoyyds'
   ```

4. WinGet 安装 Powershell；

   | `task`         | `one_command` |
   | -------------- | ------------- |
   | `run_as_admin` | `true`        |

   ```powershell
   winget install Microsoft.PowerShell
   ```

5. 安装 Chocolatey；

   | `task`         | `one_command` |
   | -------------- | ------------- |
   | `run_as_admin` | `true`        |

   ```powershell
   Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
   ```

6. 安装 Scoop；

   ```powershell
   Invoke-RestMethod -Uri https://get.scoop.sh | Invoke-Expression
   ```

7. Scoop 安装常用辅助安装工具（Git、7Zip、dark 和 innounp 等）；

   ```powershell
   scoop install git 7zip dark innounp
   ```

8. Scoop 添加常用 `bucket`；

   ```powershell
   scoop bucket add extras games nonportable
   ```

9. 解析安装目录文件并逐个安装。
